## 优化点
- 删除模型厂商item 之间的分割线
- 设置页每个模型厂商item高度增大，防止由于鼠标hover“应用”按钮出现时整个行高度增大产生闪烁效果
- 鼠标浮动到模型厂商 item时，背景色加深，背景色应有圆角
- 点击模型厂商 item 除应用按钮外的任何位置，都展开当前模型厂商
- 展开后整个模型厂商被包裹在卡片中，卡片有边框，卡片背景色设置为和鼠标浮动在item时背景颜色一致
- 展开后的可配置项与模型厂商item之间的距离应大于可配置项之间的距离
- 删除验证按钮左侧的label
- API key 输入框后的眼睛按钮放入输入框内部
- API key 输入框和模型选择框的宽度应保持一致
