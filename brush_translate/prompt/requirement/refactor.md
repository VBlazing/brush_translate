
## 需求内容
- 调整翻译任务
    - 执行翻译时不区分单词和句子，直接将翻译结果返回至 `translate_result`
    - 调整对应 prompt 及 schema
- 调整解析任务
    - 解析功能在任何情况下都支持（无论单词还是句子）
    - 点击解析按钮后使用大模型区分当前文本是单词还是句子
        - 如果是句子处理逻辑保持不变
        - 如果是单词自动将该单词的所有词性翻译插入已翻译区域下方，每个词性作为一个片段，每个词性最多包含三个释义
    - 调整对应 prompt 及 schema
