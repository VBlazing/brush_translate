## 需求内容
1. 要求模型按照 ../structure.json 文件中的 json schema 输出翻译结果

## 细节
1. 如果源语言选择了“自动检测”，按照正常流程翻译
2. 如果源语言选择了固定的语言（如英语），要求模型按照固定的语言翻译；如果模型检测到的语言类型与设置的源语言类型不一致，返回错误“语言不一致”
3. 翻译过程中的错误原因要通过 structure.json 中的 error_message 返回
4. 根据模型返回的结构，处理展示内容
    a. 翻译失败，以醒目的颜色展示 error_message
    b. 翻译成功，按照正常颜色展示 translate_result
